# Resume Analyzer

This project analyzes resumes and matches them with job descriptions.

## Setup

1. Clone the repository.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the backend:
   ```
   cd backend
   python -m flask run
   ```
4. Run the frontend:
   ```
   cd frontend
   python app.py
   ```
